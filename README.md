# 📊 Financial Report Dashboard — Power BI

## 🔍 Project Overview
An interactive Financial Report Dashboard built using Power BI, analyzing company sales, profit, COGS, and units sold data for the years 2013 and 2014.

---

## 📁 Pages in the Dashboard
| Page | Description |
|------|-------------|
| Home | Overview with KPI cards and quarterly trend line |
| Profit | Country-wise & segment-wise profit breakdown |
| Unit Sold | Product-wise units sold analysis |
| COGS | Cost of Goods Sold by segment and product |

---

## 📈 Key Insights
- 💰 Total Sales: **119M** | Profit: **16.89M** | COGS: **102M**
- 📦 Best Product: **Paseo** (highest units sold)
- 📅 **Q4** recorded the highest quarterly growth (7.5M)
- 🌍 **Germany & France** top in country-wise profit
- 🏢 **Government segment** leads with 2.9M profit
- 🎯 **Medium discount band** has the highest sales share

---

## 🛠️ Tools Used
- Microsoft Power BI
- Microsoft Excel (data source)
- DAX (Data Analysis Expressions)
